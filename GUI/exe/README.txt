這裡是GUI視窗的使用說明：

1. 在執行exe時會跳出menu視窗，進行遊戲前的選擇，可選擇的內容如下：
	- 播放音效
	- 開始遊戲，預設下為黑先 (Clicked to start the game)
	- 選擇 Player 的 先後手
	- 觀賞波奇的圖片
   此外，在button有進行顏色的特效處理。

2. 在點擊後(Clicked to start the game)，會盡如遊戲畫面，遊戲可點擊的如下：
	- 投降按鈕(Resigned)
	- 點擊進行盤面遊玩
   在進行移動的同時，console也會一併顯示盤面狀況

   在畫面顯示上，作了：
	- 當前遊戲誰先手的文字顯示介面
	- 棋盤按鈕的顏色特效，以及可走路徑的顯現
	- 遊玩的時間顯示

3. 在進行點擊投降按鈕，或是遊戲結束的時候會關閉視窗，並且在console上顯示遊戲結果
